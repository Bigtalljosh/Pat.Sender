# Release Notes

## Version 3.0

### Breaking Changes 
This now sends the body as a stream payload, as such requires that `PB.ITOps.Messaging.PatLite` must be on version `0.7.52` or later.

### Updates 
This is now compatible with .Net Standard 2.0 and .Net 4.5.2.
